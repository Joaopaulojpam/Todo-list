O sistema possui as seguintes funcionalidades:

• Cadastro de novas tarefas
• Título e descrição da tarefa
• Registro automático da data de criação
• Definição de prioridade (Baixa, Média ou Alta)
• Status da tarefa (Pendente ou Concluída)
• Listagem de todas as tarefas cadastradas
• Pesquisa de tarefas pelo título
• Marcar tarefas como concluídas
• Edição de tarefas existentes
• Exclusão de tarefas
• Validação para impedir cadastro de tarefas vazias
• Armazenamento das tarefas utilizando LocalStorage
